Example Tab

This view is served from tab.html inside the mod folder.

How to create your own tab

{ "sidebarTab": { "label": "My Tab", "html": "tab.html" } }

Add this to your mod.json. The HTML file is rendered in a sandboxed iframe — you can use any CSS or JS you like.

Reading local files from index.js

const fs = require('fs') const path = require('path') module.exports = { register(api) { const dir = api.getDir() const data = JSON.parse( fs.readFileSync(path.join(dir, 'data.json'), 'utf-8') ) api.log('Loaded ' + data.length + ' items', 'success') } }

api.getDir() returns the absolute path to your mod's folder.

Translations

Place i18n/en.json and i18n/de.json in your mod folder to provide translated labels for the sidebar and tab view. Keys tabLabel, tabTitle, tabSubtitle, tabEmpty are used when no tab.html is provided.
